# 연속된 수의 합

## 문제 설명

연속된 세 개의 정수를 더해 12가 되는 경우는 3, 4, 5입니다. 두 정수 num과 total이 주어집니다. 연속된 수 num개를 더한 값이 total이 될 때, 정수 배열을 오름차순으로 담아 return하도록 solution함수를 완성해보세요.

## 제한사항

- 1 ≤ num ≤ 100
- 0 ≤ total ≤ 1000
- num개의 연속된 수를 더하여 total이 될 수 없는 테스트 케이스는 없습니다.

## 입출력 예

| num | total | result       |
| --- | ----- | ------------ |
| 3   | 12    | [3,4,5]      |
| 5   | 15    | [1,2,3,4,5]  |
| 4   | 14    | [2,3,4,5]    |
| 5   | 5     | [-1,0,1,2,3] |

## solution

```js
function solution(num, total) {
  var answer = [];
  if (num % 2 === 1) {
    answer.push(total / num);
    for (let i = 1; i < num / 2; i++) {
      answer.push(total / num + i);
      answer.push(total / num - i);
    }
  } else {
    answer.push(Math.floor(total / num));
    answer.push(Math.ceil(total / num));
    for (let i = 1; i < num / 2; i++) {
      answer.push(Math.floor(total / num) - i);
      answer.push(Math.ceil(total / num) + i);
    }
  }
  return answer.sort((a, b) => a - b);
}
```

짝수 일 때와 홀수 일 때를 나누어서 생각하였다. 홀수 일 때는 중간값을 구하고, 중간값을 기준으로 양 옆으로 값을 구해주었다. 짝수 일 때는 중간값을 내림과 올림으로 나누어서 값을 구해주었다.

## 다른 사람 풀이

```js
function solution(num, total) {
  var min = Math.ceil(total / num - Math.floor(num / 2));
  var max = Math.floor(total / num + Math.floor(num / 2));

  return new Array(max - min + 1).fill(0).map((el, i) => {
    return i + min;
  });
}
```

최소값과 최대값을 구하고, 최소값부터 최대값까지 배열을 만들어주었다.
그 후 값 요소에 인덱스값과 최소값을 더 해 배열을 완성 시킨 모습이다.
